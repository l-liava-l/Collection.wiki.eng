`result` in a filter references to logical operation value .

### Examples

```js
$C([1, 2, 3]).every(function (el) {
	return el > 1;
}) // false

$C([1, 2, 3]).every({
	filter: function (el) {
		return el > 10;
	},

	inverseFilter: true
}) // true
```
